# Tremp.it

A carpool platform that connects drivers and passengers for recurring and one-time rides.

## Tech Stack
- Backend: Node.js + Express
- Frontend: HTML, CSS, JavaScript
- Database: PostgreSQL

## Features (planned)
- User roles: driver / passenger
- Recurring rides
- Ride subscriptions
- Seat management
- Ride requests & approvals

## Project Status
In active development
